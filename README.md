# N+ Dark Theme

A VS Code theme that combines:

* Dark Theme inspired UI and background
* Dark+ syntax highlighting
* Enhanced readability for JavaScript, TypeScript, Go, HTML, CSS, JSON, Markdown and more

## Installation

### Install from VSIX

1. Download the latest `.vsix` release.
2. Open VS Code.
3. Go to Extensions (`Ctrl + Shift + X`).
4. Click the `...` menu.
5. Select **Install from VSIX**.
6. Choose the downloaded file.

### Install from Source

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/n-plus-dark-theme.git
cd n-plus-dark-theme
```

Install VSCE:

```bash
npm install -g @vscode/vsce
```

Package the extension:

```bash
vsce package
```

Install the generated VSIX:

```text
Extensions → ... → Install from VSIX
```

## Activate Theme

1. Open Command Palette (`Ctrl + Shift + P`)
2. Run:

```text
Preferences: Color Theme
```

3. Select:

```text
N+ Dark Theme
```
